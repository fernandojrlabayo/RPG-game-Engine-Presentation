About the game
RPGMakerMV is a program that allows users to create their own role-playing video games. Most versions include a tile set based map editor (tilesets are called chipsets in pre-XP versions), a simple scripting language for scripting events, and a battle editor. All versions include initial premade tilesets, characters, and events which can be used in creating new games. One feature of the PC versions of RPG Maker programs is that a user can create new tilesets and characters, and add any new graphics the user wants.
